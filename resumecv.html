AI Writing Marker - a personal project | February 2025 – Present
While working as a tutor at Qs School, I developed a Chrome extension to help teachers mark student work 250% faster and more accurately. This extension is used by tutors to reduce essay marking times from 5 minutes to 2 minutes and marks almost 1000 essays per week. The extension was built using React, driven by a back-end API that runs as AWS Lambda functions. The back-end was built using NodeJS and Express. It can also be hosted in Azure App Service as well as in Heroku. The Chrome extension and the back-end integrates with Auth0 for OIDC user authentication and sign-up. The back-end utilises OpenAI API for marking the student work and passes the results to the front-end to annotate the student's essays with corrections, comments, and rubric ratings.
