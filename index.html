Key Technologies & Tools Used

This website has been built with the following technologies and tools:


  • Frontend Framework: Next.js (for its SEO benefits and fast performance). I chose Next.js for its server-side rendering and SEO benefits, compared to ReactJS which is more client-side focused.
  • Styling: Tailwind CSS (for clean and responsive UI). Compared to CSS-in-JS solutions (like styled-components). Tailwind CSS offers utility-first styling and flexibility.
  • CI/CD Pipeline: GitHub Actions for automatic linting, testing, and deployment
  • Hosting: AWS S3 (for reliable and cost-effective hosting)
  • Developer Experience: I have also set up ESLint, Prettier, and Git hooks using Husky for enhanced development workflows, ensuring code quality and consistency.

AI Writing Marker

  • While working as a tutor at Qs School, I developed a Chrome extension to help teachers mark student work faster and more accurately. The extension was built using React, driven by a back-end API that runs as AWS Lambda functions. The back-end was built using NodeJS and Express. It can also be hosted in Azure App Service as well as in Heroku. The Chrome extension and the back-end integrates with Auth0 for user sign-up and authentication. The back-end utilises OpenAI API for marking the student work and passes the results to the front-end to annotate the student's essays with corrections, comments, and rubric ratings.
